⚠️ Important

Make sure to restart your terminal after installation to ensure the lupo command is available.

Quick Start

Follow these steps to get started with LUPO in under 5 minutes.

Step 1: Initialize

Create a new LUPO project:

lupo init my-project cd my-project

Step 2: Configure

Set up your basic configuration:

lupo config set api-key YOUR_API_KEY lupo config set environment production

Step 3: Run

Start your first LUPO process:

lupo start

That's it! Your LUPO instance is now running. Visit http://localhost:3000 to see it in action.

Configuration

Customize LUPO to fit your specific needs with our flexible configuration options.

Configuration File

LUPO uses a lupo.config.json file for configuration:

{ "apiKey": "your-api-key", "environment": "production", "features": { "featureA": true, "featureB": false }, "limits": { "maxConnections": 100, "timeout": 30000 } }

Environment Variables

You can also configure LUPO using environment variables:

  • LUPO_API_KEY - Your API key
  • LUPO_ENV - Environment (development/production)
  • LUPO_PORT - Server port (default: 3000)
