LUPO API

Integrate LUPO's visual AI capabilities into your applications with our powerful API. Currently in developer preview for select partners.

Quick Start

Get started with the LUPO API in minutes

cURL 
# Authenticate with your API key
curl -X POST https://api.lupo.ai/v1/analyze \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "image": "base64_encoded_image",
    "prompt": "What do you see in this screenshot?",
    "context": "user_interface"
  }'
